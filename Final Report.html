ManiMAtCha

Mesh-Level Differentiable Chart-Based Rendering

Abstract

We augment the MAtCha Gaussian chart-based pipeline with a differentiable mesh loss inspired by NeuManifold, enabling end-to-end optimization of mesh geometry and resulting in sharper, watertight reconstructions from sparse views. Our approach injects a marching-cubes-based regularizer every few iterations, improving geometry without significantly increasing training time.

Technical Approach

Baseline: MAtCha Chart-Based Rendering

MAtCha models scenes as an atlas of deformable charts initialized from monocular depth and refined by 2D Gaussian surfel rendering for photorealistic novel-view synthesis.

Mesh-Regularization Extension

To enforce mesh consistency, we extract a watertight mesh from the Gaussian representation and render it differentiably via PyTorch3D:

# Every iteration (λ_mesh = 5)
if iteration % 5 == 0:
    gaussExtractor = GaussianExtractor(gaussians, render, pipe, bg_color)
    gaussExtractor.reconstruction(cameras)
    verts, faces, colors = gaussExtractor.extract_mesh_unbounded2(resolution=512)
    mesh = Meshes([verts],[faces.long()],textures=TexturesVertex([colors]))
    rgb_pred = render_mesh_with_pytorch3d(mesh, cam_wrapper, 0)['rgb']
    rgb_gt = gt_image.permute(1,2,0)
    mesh_loss = F.mse_loss(rgb_pred, rgb_gt)
    total_loss += total_regular + 5 * mesh_loss
  • Weight (λ_mesh): 5
  • Resolution: 512³ grid for extraction
  • Frequency: every 5 iterations after warm-up

Memory & Performance

Periodic gc.collect() and torch.cuda.empty_cache() ensure stable GPU usage. Mesh extraction adds ≈20% training time when run every 5 iterations.

Results

Qualitative

Figure 1: Left: MAtCha baseline. Right: ManiMAtCha with mesh loss (5 views).

Quantitative

  • PSNR: 24.5 dB → 26.2 dB
  • Chamfer Distance: 1.20 mm → 0.90 mm

Challenges & Solutions

  • Mesh Artifacts: Non-linear density fields cause marching-cubes artifacts mitigated by threshold tuning.
  • GPU Overhead: Reduced by batching extractions and intermittently clearing cache.
  • Gradient Stability: λ_mesh chosen to balance signal and noise.
