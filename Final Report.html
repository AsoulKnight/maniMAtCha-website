Mesh-Level Differentiable Chart-Based Rendering

Abstract

We augment MAtCha’s Gaussian chart pipeline with a differentiable mesh loss inspired by NeuManifold, sharpening reconstructions from sparse views with minimal overhead.

Technical Approach

We build on MAtCha’s chart-based reconstruction by freezing charts after SfM alignment and applying a mesh-based loss to refine only the Gaussian primitives.

Mesh-Regularization Extension

Every 5 iterations (λmesh = 5), we:

  1. Extract mesh: verts, faces, colors = extractor.extract_mesh_unbounded2(resolution=512)
  2. Render mesh: rgb_pred = render_mesh_with_pytorch3d(mesh, cams, 0)['rgb']
  3. Compute loss: mesh_loss = MSE(rgb_pred, rgb_gt)
  4. Backpropagate: total_loss += total_reg + 5 * mesh_loss

Implementation Challenges

  • Memory & computation heavy: mitigated by lower resolution and cache clearing.
  • Hole artifacts: planning lightweight Gaussian coverage regularizers.

Lessons Learned

  • Decoupling charts and Gaussian refinement simplifies design but risks misalignment.
  • Mesh loss boosts accuracy but adds overhead; effective batching and memory management are crucial.
